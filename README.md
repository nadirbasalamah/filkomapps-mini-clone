# filkomapps-mini-clone

An API contains many functionalities that identical with filkomapps.

## Technology Stack:

The technology stack that used in this API are:

- NodeJS
- express
- express-validator
- jsonwebtoken
- mongoose
- bcryptjs
- config
- nodemon

## Features

The features that available in this API are:

1. For college student:

- Upload a proposal or final report.
- Upload certain documents for final assignment.

2. For lecturer:

- View or verify a proposal or final report from college student.
- Change the progress status of assignment for college student.
